# Desafio Cypress - Automação UI e API

Projeto desenvolvido como parte do desafio técnico de automação utilizando **Cypress** com **Cucumber**, contemplando testes de **UI** e **API**.

---

## 🛠️ Tecnologias utilizadas

- Cypress
- Cucumber
- JavaScript
- Node.js

---

## ✅ Pré-requisitos

- Node.js instalado
- NPM instalado

---

## ▶️ Como executar o projeto

1. Instale as dependências:

```bash
npm install
```

2. Execute o Cypress em modo interativo:
```bash
npx cypress open
````

## Evidências

As evidências de execução (prints e vídeos) são geradas ao executar o projeto em modo headless:

```bash
npx cypress run
``
